# Task Manager - MERN Stack

## Project Overview

Task Manager is a Full Stack Web Application built using MongoDB, Express.js, React.js, and Node.js.

The application allows users to register, login, create tasks, view tasks, update tasks, and delete tasks.

## Features

* User Registration
* User Login Authentication
* Create Tasks
* View Tasks
* Update Tasks
* Delete Tasks
* MongoDB Atlas Database
* REST API Integration

## Technologies Used

### Frontend

* React.js
* Axios
* React Router DOM

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas
* Mongoose

## Installation

### Backend

```bash
cd server
npm install
node server.js
```

### Frontend

```bash
cd client
npm install
npm start
```
